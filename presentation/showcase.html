05 · 结果总结

实验结果与后续计划

实验结果

  • 高效：热启动 < 0.5 秒，三层缓存支撑
  • 智能检索：支持自然语言 / 中文，OpenAlex 智能选题
  • 易用：零安装，浏览器即可访问
  • 可扩展：模块化解耦，便于迭代
  • 实用：覆盖科研 / 学习 / 技术跟进

问题与改进方向

  • 解析失败 → 升级配置 + 格式兜底
  • 冷启动慢 → 预热常用关键词缓存
  • 摘要串行 → Map 阶段并发，进一步提速
  • 摘要偶漏要点 → 优化章节权重
