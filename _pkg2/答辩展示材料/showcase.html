03 · 技术亮点

检索增强 · 批处理 · 并发

中文 / 自然语言检索

中文一句话经 LLM 改写为英文检索式；已是干净英文或精确标题则原样保留，避免被改散。

智能选题 + 去重合并

OpenAlex 按相关性 + 引用排序；同一篇的预印本/正式版自动合并，优选 arXiv 直链。

按意图自动定篇数

精确论文标题→少而精，宽泛话题→多给候选，无需手动调参。

arXiv 标题兜底

出版商链下载不动时，用标题回查 arXiv 取全文；结果落盘缓存、快速失败、反爬域名跳过。

MinerU 批量解析

多篇合并为一次目录模式调用，模型只装载一次；批量失败自动回退逐篇，稳健。

摘要 Map 章节级并发

逐章节摘要改线程池并发，顺序严格保持，实测约 2.6× 提速；配合三层缓存热启动 0.4 秒。

05 · 结果总结

实验结果与后续计划

实验结果

  • 高效：热启动 < 0.5 秒，三层缓存支撑
  • 智能检索：支持自然语言 / 中文，OpenAlex 智能选题
  • 易用：零安装，浏览器即可访问
  • 可扩展：模块化解耦，便于迭代
  • 实用：覆盖科研 / 学习 / 技术跟进

问题与改进方向

  • 解析失败 → 升级配置 + 格式兜底
  • 冷启动慢 → 预热常用关键词缓存
  • 摘要串行 → Map 阶段并发，进一步提速
  • 摘要偶漏要点 → 优化章节权重
